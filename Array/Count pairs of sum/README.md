# Count Pairs with Given Sum
Given an array `arr[]` and an **integer target**. You have to find numbers of pairs in array `arr[]` which sums up to given **target**.
## Prerequisite
```java
import java.util.Scanner;
import java.util.HashMap;
```
### Example:
```
Input  : [1, 5, 7, -1, 5]
         target = 6 
Output : 3
```
